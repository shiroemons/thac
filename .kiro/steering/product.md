# Product Overview

Better-T-Stackを基盤としたモダンなフルスタックWebアプリケーション。React、TanStack Start、Honoを組み合わせた型安全なTypeScriptスタックで構築されている。

## Core Capabilities

- **型安全なフルスタック開発**: TypeScriptによるエンドツーエンドの型安全性
- **SSR対応フロントエンド**: TanStack Startによるサーバーサイドレンダリング
- **高性能API**: Honoによる軽量で高速なバックエンドAPI
- **認証システム**: Better-Authによるセキュアな認証機能
- **モノレポ構成**: Turborepoによる効率的なビルドとパッケージ管理

## Target Use Cases

- モダンなWebアプリケーションの構築
- 型安全性を重視したチーム開発
- SSRが必要なパフォーマンス重視のアプリケーション

## Value Proposition

- **開発者体験**: TypeScript + Bun + Biomeによる高速で快適な開発環境
- **モジュラー設計**: 共有パッケージ（auth、db）による関心の分離
- **スケーラビリティ**: モノレポ構成による将来の拡張性
