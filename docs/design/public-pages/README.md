## 概要

**東方編曲録 〜 Arrangement Chronicle** の一般公開ページの設計ドキュメント。

### プロジェクト情報

| 項目 | 内容 |
|------|------|
| サイト名 | 東方編曲録 〜 Arrangement Chronicle |
| コピーライト | © 2025 迷い家の白猫 |
| デザインテーマ | モダン・クリーン |
| 対応デバイス | デスクトップ、タブレット、モバイル |

### 設計方針

- **モバイルファースト**: 小画面から設計し、大画面に拡張
- **アクセシビリティ**: WCAG AA準拠を目標
- **パフォーマンス**: 高速な体感を重視
- **一貫性**: 共通コンポーネントの再利用

## クイックリファレンス

### ナビゲーション構造

```
ホーム（/）
├── 原曲（/original-songs）
│   ├── 公式作品一覧（/official-works）
│   └── 公式曲詳細（/original-songs/:id）
├── サークル（/circles）
│   └── サークル詳細（/circles/:id）
├── アーティスト（/artists）
│   ├── アーティスト詳細（/artists/:id）
│   └── 役割一覧（/roles）
├── イベント（/events）
│   └── イベント詳細（/events/:id）
├── 統計（/stats）
├── 検索（/search）
└── About（/about）
```

### 主要フィルター

| ページ | フィルター |
|--------|-----------|
| サークル一覧 | 記号・数字 \| 英字 \| かな \| 漢字 |
| アーティスト一覧 | 記号・数字 \| 英字 \| かな \| 漢字 + 役割（編曲/作詞/Vo） |
| イベント一覧 | シリーズ別 \| 年別 |

### 表示切替

- **一覧ページ**: グリッド（⊞）/ リスト（≡）
- **状態保存**: localStorage
- **URL同期**: フィルター状態をURLパラメータで共有可能
